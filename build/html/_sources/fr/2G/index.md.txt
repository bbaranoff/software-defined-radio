# 2G : GSM/GPRS/EDGE

**2G** (Deuxième Génération) désigne la **première génération numérique de la téléphonie mobile**, apparue dans les années 1990. Elle remplace les réseaux analogiques (1G) par des standards numériques comme le **GSM** (Global System for Mobile Communications).

**Points clés :**

* Numérique (et non analogique) : appels plus clairs, moins de parasites, chiffrement basique (A5/1, A5/2)
* Introduction des SMS (textos) et données de base (GPRS/EDGE)
* Débits typiques : 10–100 kbps
* Principaux standards : **GSM** (monde), **CDMA** (États-Unis/Asie)
* Encore utilisée pour voix/SMS, mais remplacée progressivement par la 4G/5G

**Résumé :**
2G = mobile numérique, appels + SMS + données basiques, socle des réseaux mobiles modernes.
